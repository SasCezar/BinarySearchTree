Detailed Description

template<typename T, typename compT>
class bst< T, compT >

Class that implement a binary search tree

The class requires to specify the type of the values The class requires a comparator for the specified values, the comparator have to return an value of type int that equals to one of these:

  • 0 : if the values are equal
  • 1 : if value A is greater than value B
  • -1: if value B is less than value B
